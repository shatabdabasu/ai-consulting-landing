# AI Consulting Service Page

A professional single-page landing website for an AI consulting business, built as part of a technical evaluation task.

## Overview
This project showcases a clean, responsive AI service landing page designed to present consulting offerings clearly and professionally. The focus is on usability, clarity, and real-world frontend practices rather than experimental features.

## Features
- Professional AI consulting landing page
- Fully responsive design (desktop and mobile)
- Service sections with visual backgrounds
- Working contact form for user inquiries
- Clean and minimal UI suitable for client-facing use

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub Pages (deployment)

## AI Assistance
ChatGPT was used to accelerate initial layout planning, content drafting, and form-handling logic. All design, structure, and final refinements were manually implemented and validated.
